\section{System Architecture Principles}
\begin{itemize}
    \item Layered architecture separates concerns across distinct 
    system components
    \item Separation of concerns improves maintainability, 
    testability, and modularity
    \item Client-server architecture separates user interaction 
    from backend logic
    \item REST APIs enable communication between frontend 
    and backend over HTTP
    \item Vector databases support efficient storage and retrieval 
    of embeddings at scale
\end{itemize}
