\section{Related Work and Existing Tools}
\begin{itemize}
    \item Several tools and frameworks exist for automated or 
    assisted machine learning method selection
    \item AutoML tools such as Auto-sklearn and TPOT automate 
    model selection and hyperparameter tuning, but treat selection 
    as an optimization problem rather than a knowledge-driven process
    \item Algorithm selection frameworks such as meta-learning 
    approaches recommend methods based on dataset characteristics, 
    but typically lack domain context and literature grounding
    \item Knowledge-based and ontology-driven systems have been 
    proposed for ML method selection, but few combine ontologies 
    with semantic literature retrieval
    \item Existing tools generally do not provide implementation 
    support or links to supporting research literature
    \item MLguide is positioned as a decision-support tool that 
    combines ontology-based filtering, semantic retrieval, and 
    implementation support in a single system
\end{itemize}
