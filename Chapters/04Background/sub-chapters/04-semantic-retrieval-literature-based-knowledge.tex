\section{Semantic Retrieval and Text Similarity}
\begin{itemize}
    \item Text can be represented as embeddings (vectors)
    \item Similarity measures can compare meaning between texts
    \item Used to match problem descriptions with relevant content
    \item More flexible than keyword-based search
\end{itemize}

\textbf{Literature-Based Discovery / Retrieval}
\begin{itemize}
    \item Scientific articles are a rich but unstructured knowledge source
    \item Challenges include large volume, noise, and unclear relevance
    \item Methods can be linked to articles through extraction and metadata
    \item Semantic retrieval can improve matching between problems and literature
\end{itemize}
